Usage notes. Keep aria-hidden="true" on the monogram when it sits next to the wordmark so screen readers don't announce “jb joshua Briley.” Wrap the lockup in an anchor with group and add group-hover:scale-105 transition-transform on the mark for the standard nudge. The coral accent on Briley is required — never render the surname in the same color as the first name.
